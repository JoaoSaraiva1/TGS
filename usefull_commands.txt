To retrieve the cluster IP of the pg-admin-pgadmin service, you can run the following command:
kubectl get svc --namespace default pg-admin-pgadmin -o jsonpath='{.spec.clusterIP}'

Verificar o estado dos pods:
kubectl get pods --namespace default

Ver o que se passa com o pod:
kubectl describe pod <pod_name> --namespace default

Ver logs do pod:
kubectl logs <pod_name> --namespace default

Port-forward
kubectl port-forward svc/<service_name> 8080:80 --namespace default

Para obter informações da BD
kubectl describe statefulsets postgresql-db 

Obter url para um service
minikube service <service_name> --url
